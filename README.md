# Ejercicios_Practica_1DAW

En este repositorio se irán subiendo ejercicios de programación aplicando uno o varios paradigmas. Cada paradigma o conjunto de paradigmas estará en una carpeta distinta.

Los ejercicios pueden estar subidos en Lenguaje_DAW o en C# (Lenguaje_DAW: lenguaje inventado por el profesor de Programación que combina elementos de C#, Java y Python. Diseñado para tener una introducción sencilla a la programación.).
